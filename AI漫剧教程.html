1爆款脚本极速生成

▼

方法1：经典IP魔改法 ⭐

借用大众熟知的电影/电视剧/动漫的经典剧情框架，替换成你的专属人设，既省脑子又自带爆款基因

✅ 直接复制指令模板（改括号里的内容就行）
你是资深AI漫剧脚本师，擅长写1分钟短视频脚本，节奏快、冲突强、结尾留钩子。 请基于《僵尸先生》开篇1分钟的经典剧情框架，生成一个全新的1分钟分镜脚本，要求： 1. 角色替换：主角=橘猫九叔，徒弟=小老鼠文才，僵尸王=黑猫，女鬼=白猫，新增角色=黄狗保安队长 2. 时长：严格控制在60秒，共8-10个分镜，每个分镜标注【时长+镜头语言】 3. 内容：保留原片"义庄守夜+僵尸异动"的核心情节，加入动物角色的萌系反差感 4. 台词：简洁口语化，每句不超过15字，标注对应角色 5. 结尾：第58秒留钩子，比如"黑猫僵尸突然睁开了金色的眼睛"
💡 进阶优化技巧：
  • 可替换IP：《功夫》《大话西游》《甄嬛传》《猫和老鼠》《西游记》（优先选无版权争议的老IP）
  • 人设差异化：把人类换成动物/水果/文具/表情包，越反差越容易火（如甄嬛传角色换成猫咪）
  • 批量生成：一次让豆包生成3个不同版本，挑最好的再让它"加强第3、7个分镜的冲突"

方法2：爆款视频反推改编法 🔥

这是新手最快出爆款的方法，没有之一！ 不用自己想剧情，直接抄爆款的骨架，换皮成你的内容。

第一步：精准反推爆款提示词

复制抖音/快手的爆款视频链接，发给豆包：

请根据这条视频内容，反推可直接用于AI视频生成的完整分镜提示词，必须严格涵盖以下所有要素，每个要素写详细： 1. 主体：每个人物/动物的外貌、服装、表情、体型特征 2. 动作：精确到每个肢体动作、面部表情变化 3. 场景：环境细节、整体氛围、时间（白天/黑夜/黄昏） 4. 光影：光线类型（自然光/灯光/月光）、强度、照射方向 5. 运镜：推/拉/摇/移/俯拍/仰拍/特写，标注每个镜头的运镜方式 6. 风格：视觉风格、整体色调、画风 7. 画质：4K、60fps、电影级质感、无模糊、无畸变 8. 转场：每个分镜之间的转场方式
第二步：一键改编成原创故事

把豆包反推出来的完整提示词复制，再发给豆包：

你是资深AI胖橘治愈视频创作者（根据你的赛道替换：AI漫剧/悬疑动画/萌宠动画/古风动画），擅长写脚本和改编故事。 请严格保留上面这个脚本的分镜结构、节奏、冲突点和时长，编写一个情节、人物、场景完全不同，但题材和内核类似的全新原创故事。 要求：角色全部换成胖橘和它的朋友们，风格治愈搞笑，结尾留一个让人想看下一集的钩子。
🔧 工具补充：
  • 有Gemini的用户：直接上传视频文件，发上面第二步的指令，反推+改编一步到位
  • 没有链接的情况：把爆款视频文案复制给豆包，加上"请根据这段文案还原成完整的分镜脚本"

✅ 脚本二次打磨（必做！）

生成脚本后，再发给豆包一句话优化：

请把上面的脚本改成适合即梦AI生成视频的格式，每个分镜单独成段，开头标注【X秒-X秒】，去掉多余的文学描述，只保留AI能识别的具象化内容。

风格一致性控制技巧

  1. 把风格提示词复制粘贴到每一个分镜的最后，确保全片风格统一
  2. 生成第一个分镜后，把满意的截图作为参考图，后续所有分镜都用这张图作为风格参考
  3. 不要在不同分镜里更换风格关键词，否则会出现画风突变

3参考图提示词+用法

▼

参考图提示词公式

① 人物参考图提示词

角色身份 + 外貌特征 + 服装 + 表情 + 姿势 + 风格 + 画质

示例：

橘猫九叔，穿着黄色道袍，戴着八卦帽，留着白色胡子，表情严肃，双手背在身后，吉卜力画风，4K，高清正面全身照
② 场景参考图提示词

场景名称 + 环境细节 + 时间 + 天气 + 风格 + 画质

示例：

古代义庄，破旧的木质房屋，摆放着棺材，墙上贴着黄色符咒，夜晚，月光透过窗户照进来，阴森氛围，中国风，4K

参考图进阶用法（新手必学）

🔒 人物一致性锁定
  • 先用AI生成一张你满意的主角正面全身照，保存下来
  • 后续所有分镜生成时，都上传这张图作为参考图，参考强度调为 0.7~0.8
  • 提示词里加上"和参考图中的人物完全一致，外貌、服装、体型不变"
🏠 场景一致性锁定
  • 先生成主场景的全景图，作为固定参考图
  • 同一场景的不同镜头，都用这张全景图作为参考，只改变运镜和人物位置
🎨 多参考图融合
  • 最多上传 3张参考图，分别对应人物、场景、风格
  • 比如：1张主角图 + 1张场景图 + 1张风格图，AI会自动融合三者的特征

4声音固定与制作

▼

方法1：即梦自带音频提取法 ✨ 最方便

  1. 先用即梦生成一段完整的剧情视频
  2. 用剪映导入这段视频，点击"音频分离"，提取所有角色配音
  3. 把每个角色的配音剪成单独片段，重命名保存（如"橘猫九叔-台词1"）
  4. 后续所有视频，都用这些提取出来的音频片段，直接导入剪映对应到分镜即可

方法2：抖音AI音频提取法 🎤 音色丰富

  1. 在抖音搜索"AI配音"，找到你喜欢音色的视频（优先选明确标注AI生成的）
  2. 复制视频链接，用"轻抖"小程序提取无水印视频
  3. 导入剪映，分离音频，截取需要的片段保存
⚠️ 版权提示：绝对不要提取真人演员的原声，只提取AI生成的配音，避免版权纠纷！

方法3：TTS固定音色生成法 🎯 最稳定

  1. 打开豆包的"语音合成"功能，选择喜欢的音色
  2. 把脚本里的台词逐句复制进去，生成音频
  3. 同一个角色永远用同一个音色，不要更换
  4. 进阶：用即梦的"音色克隆"功能，克隆专属音色，只需30秒语音样本

音频后期加分技巧 🎵

  • 加合适的背景音乐（剪映免费音乐库），音量调为 -20 ~ -15，不要盖过人声
  • 加音效：开门声、脚步声、风声等提升沉浸感（剪映音效库免费）
  • 统一全片音量：选中所有音频，点击"音量统一"，避免忽大忽小

5标准化文件夹管理

▼

📁 统一命名规则

日期 - 项目名称 - 集数

示例：20240520-橘猫九叔-第1集

📂 固定文件夹结构（每个项目都建这5个文件夹）

📂 20240520-橘猫九叔-第1集 ├─ 📝 脚本 │ ├─ 初稿.txt │ ├─ 优化后分镜脚本.txt │ └─ 最终提示词.txt ├─ 🖼️ 参考图 │ ├─ 人物参考 │ │ ├─ 橘猫九叔.png │ │ └─ 小老鼠文才.png │ ├─ 场景参考 │ └─ 风格参考 ├─ 🎵 音频 │ ├─ 配音 │ ├─ 背景音乐.mp3 │ └─ 音效 ├─ 🎬 生成素材 │ ├─ 分镜1.mp4 │ └─ 分镜2.mp4 └─ ✅ 成品 └─ 橘猫九叔第1集-最终版.mp4

好习惯 ✅

  • 每做完一个项目，就把整个文件夹归档备份到云盘
  • 所有文件都用中文命名，不要用数字或字母缩写，避免后期找不到

6豆包反推提示词终极版

▼

🚀 通用终极反推指令（直接复制，不用改）

请深度分析这条视频的每一个镜头，反推可用于即梦AI生成的逐帧分镜提示词，要求： 1. 严格按照【镜头序号+时长+运镜+主体+动作+场景+光影+风格+画质】的格式输出 2. 主体描述精确到五官、服装、配饰、表情、体型，动作描述精确到肢体的每一个部位 3. 场景描述包含所有可见的背景细节、物品摆放、环境氛围 4. 光影描述包含光线来源、颜色、强度、照射方向、阴影效果 5. 运镜描述包含镜头运动速度、起止位置、景别（全景/中景/近景/特写） 6. 标注每个镜头之间的转场方式（硬切/淡入淡出/滑动/缩放） 7. 所有描述都是具象化的、AI能识别的，不要有抽象的文学描述 8. 最后单独总结这条视频的整体风格、色调和节奏特点

反推结果优化技巧 💡

  1. 反推完成后，让豆包"把上面的提示词简化，去掉重复内容，保留核心要素"
  2. 把整体风格总结复制下来，粘贴到每一个分镜的提示词最后
  3. 把反推的人物和场景描述，改成你自己的原创角色和场景

7新手必看避坑指南

▼

🛡️ 版权避坑

  • 不要用有版权的IP形象（比如迪士尼、漫威）
  • 不要用真人明星的脸生成视频
  • 不要用未经授权的音乐和真人配音

🤖 AI生成避坑

  • 单个分镜时长一般固定 12~15秒（12秒是随便时长，15秒是即梦时长）
  • 不要让AI生成复杂的手部动作和多人互动，容易变形
  • 生成不满意就多生成几次，或者微调提示词里的动作描述

📈 流量避坑

  • 视频时长控制在 50~60秒，完播率最高
  • 开头 3秒必须有冲突或亮点，留住观众
  • 结尾一定要留钩子，引导用户看下一集
